BookManager v2.6 


Умеет показывать/закрывать/переключать книги/жаву.
Умеет показывать/закрывать/переключать/показывать_копирайты эльфов.
В первой вкладке на цифровых клавишах - быстрый запуск для штатных менюх.
Во второй - быстрый запуск эльфов.
"#" - свернуть всё.
долгий "#" (в первой вкладке) - закрыть всё.



bookman.ini теперь переехал в $DIR_INI$/bookman/

Описание ключиков в ини файле.

отображение имён:

  что_видим : что_хотим_видеть

  Standby: Режим ожидания
  Desktop: Главное меню


[S_KEY0] - быстрый запуск по короткому нажатию "0" в первой вкладке.
[L_KEY0] - быстрый запуск по долгому нажатию "0" в первой вкладке.

[S_KEY0]: MenuItem_Bt_Activation

Запуск java-приложения:

Пишем по маске: java:Name//Vendor
Name - Название приложения
Vendor - Поставщик приложения
Оба параметра можно посмотреть в "Информации" у приложения

[S_KEY0]: java:Jimm//Jimm Project
 
[ES_KEY0] - быстрый запуск эльфа по короткому нажатию "0" во второй вкладке.
[EL_KEY0] - быстрый запуск эльфа по долгому нажатию "0" во второй вкладке.
  Пути к эльфам относительно ZBin.
  Если на катре и в телефоне присутствуют эльфы с одинаковыми именами, запущен 
    будет из памяти телефона."/" в начале - обязателен.

[ES_KEY0] /HeapMonitor.elf
[ES_KEY1] /HelloWorld/HelloWorld.elf

[START_KEY] - код клавиши вызова менеджера
[START_KEY]: 0x1F;

[KEY_MODE] - режим клавиши вызова менеджера(нажатие/отпускание/....)
[KEY_MODE]: 0x03;

[UIEE] - Разрешить/запретить закрытие менеджера при неактивности пользователя
[UIEE]: 0x01;

[EIBL] - Отображать или нет в первой вкладке эльфы с гуи-интерфейсом;
[EIBL]: 0x01;

[NBAS] - Называть книги как сессии, к которой они относятся, или брать название самих книг
[NBAS]: 0x00;

[SOT] - поднимать StandbyBook вверх списка после вызова быстрой ссылки. Для владельцев w800 обязательно должно стоять 0x00, чтобы работали ярлыки
[SOT]: 0x00;

[IGNORE_KEYLOCK] - Разрешить/запретить вызов менеджера при заблокированной клавиатуре.
[IGNORE_KEYLOCK]: 0x00;


Иконки.
Иконки должны лежать в папке $DIR_INI$/bookman/

[ICON00] - для неактивной первой вкладки
[ICON01] - для активной первой вкладки
[ICON02] - для неактивной второй вкладки
[ICON03] - для активной второй вкладки

[ICON00]: 1.png
[ICON01]: 3.gif
[ICON02]: 4.gif
[ICON03]: package[1].gif
