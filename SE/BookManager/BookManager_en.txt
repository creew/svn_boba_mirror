BookManager v2.6 

Abilities:
- display/close/switch book/java applications
- show/switch/show copirights of elfs

On the first tab you can run shortcut by digital keys.
On the second tab you can run elfs by digital keys.

"#" - hide all.
Long "#" (on the first tab) - close all.


Place file "bookman.ini" in $DIR_INI$/bookman/


Meanings of parameters in "bokman.ini".

Name displaying:

  what_see : what_want_to_see

e.g.:
  Desktop: Main menu


[S_KEY0] - Shortcut on "0" key on the first tab.
[L_KEY0] - Shortcut on long "0" key on the first tab.

e.g.:
[S_KEY0]: MenuItem_Bt_Activation

Java running:

Mask: java:Name//Vendor
Name - Midlet Name
Vendor - Midlet Vendor
You can get this params in "Information" of midlet

e.g.
[S_KEY0]: java:Jimm//Jimm Project
 
[ES_KEY0] - Run elf on "0" key on the second tab.
[EL_KEY0] - Run elf on long "0" key on the second tab.

You need to write Elfs paths relative to ZBin folder.
If you have elfs with the same name on card and in phone memory, elf from phone memory will be run.
"/" in the bigining is obligatory.

e.g.:
[ES_KEY0] /HeapMonitor.elf
[ES_KEY1] /HelloWorld/HelloWorld.elf

[START_KEY] - key code to start BookManager
[START_KEY]: 0x1F;

[KEY_MODE] - mode of start key(press/release/....)
[KEY_MODE]: 0x03;

[UIEE] - Enable/disable BookManager closing when user is inactive
[UIEE]: 0x01;

[EIBL] - Display or not elfs with GUI on the first tab
[EIBL]: 0x01;

[NBAS] - Name books as session, they are belong, or display name of books
[NBAS]: 0x00;

[SOT] - move StandbyBook on the top after shortcut run
[SOT]: 0x00;

[IGNORE_KEYLOCK] - Enable/disable BookManager start when keyboard is locked.
[IGNORE_KEYLOCK]: 0x00;


Icons.
Place icons in folder $DIR_INI$/bookman/

[ICON00] - inactive first tab
[ICON01] - active first tab
[ICON02] - inactive second tab
[ICON03] - active second tab

e.g.:
[ICON00]: 1.png
[ICON01]: 3.gif
[ICON02]: 4.gif
[ICON03]: package[1].gif


Default configuration:

[START_KEY]: 0x1F;
[KEY_MODE]: 0x03;
[UIEE]: 0x01;
[EIBL]: 0x01;
[NBAS]: 0x00;
[IGNORE_KEYLOCK]: 0x00;
[SOT]: 0x00;
[NBAS]: 0x00;
