GPView - просмотрщик картинок GPF v0.9RC1 (0xC605)
(с) Kibab

***Возможности***
Интеграция в Проводник с помощью ассоциации в extension.cfg
Открытие и показ любых GPF/GPN, даже не подкючённых в настоящий момент, заблокированных итд...
По левой софт-кнопке раскрывается традиционное меню :)
Кнопки 4, левый джой / 6, правый джой - перелистывание картинок в GPN-пакете. Для GPF отключены :).
Кнопка 5 - показать, какая картинка с таким номером используется в настоящий момент.
Показ служебной информации - десятичного номера картинки и физического размера, имени файла.

***Установка***
Скопировать эльф в папку 4:\ZBin\Kibab
Отредактировать свой extension.cfg либо взять готовый из архива.
Перезагрузиться.
После этого тыкаться в Проводнике в любой GPF/GPN.
